# NXTGN-apps
Gateway is a time-limited experimental mobile application designed to provide synchronized monitoring and status updates.
Gateway App
Gateway is a specialized, time-limited mobile application designed for synchronized monitoring. This project functions in tandem with the Gateway web portal.
Features
Synchronized Monitoring: Seamlessly connects with the Gateway web platform for accurate data relay.
Time-Limited Access: Designed for a specific one-month project lifecycle.
Lightweight Design: Optimized for efficient performance on Android devices.
Installation
Since this application is distributed independently, please follow these steps to install:
Navigate to the [Releases] section on this repository page.
Download the latest gateway.apk file.
On your Android device, open the downloaded file.
If prompted, go to your device Settings and enable "Install unknown apps" for your browser or file manager.
Proceed with the installation.
